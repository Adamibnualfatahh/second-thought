**Pause. Reflect. Decide.**

SecondThought is a mindful decision-making companion designed to help you overcome impulse actions. By introducing a deliberate pause between an impulse and an action, we help you regain control over your choices.

![SecondThought Preview](https://picsum.photos/800/400)

## ‚ú® Features

*   **Impulse Capture**: Quickly categorize your impulse (Shopping, Message, Work, Feeling, etc.).
*   **Forced Delay**: Choose a waiting period (from 5 minutes to 24 hours) to let the emotional intensity subside.
*   **Guided Reflection**: Answer simple prompts to understand the "why" behind your impulse.
*   **Mindful Timer**: A calming waiting screen with visual breathing aids to help you relax.
*   **Notification Reminders**: Get notified gently when your reflection time is up.
*   **Decision Journal**: Record your final decision to track your growth over time.

## üöÄ Getting Started

### Prerequisites

*   Node.js (v16 or higher)
*   npm or yarn

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/Adamibnualfatahh/second-thought.git
    cd second-thought
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

3.  Run the development server:
    ```bash
    npm run dev
    ```

4.  Open your browser and visit `http://localhost:5173`.
